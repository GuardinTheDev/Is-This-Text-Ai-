# Is-This-Text-Ai-
Girilen metnin ai olup olmadığını tahmin eden algoritma
Modelin eğitiminde Hugging Face üzerinde açık kaynak olarak paylaşılan Kaggle DAIGT V2 veri seti kullanılmıştır.

Veri Seti Linki: Kaggle Comp DAIGT

İçerik: Çeşitli konularda insanların yazdığı denemeler (essays) ve ChatGPT (GPT-3.5/GPT-4) gibi modeller tarafından üretilen yapay zeka metinleri.

Veri Dağılımı: Veri seti eğitim esnasında otomatik olarak %80 Eğitim (Train) ve %20 Test (Evaluation) olacak şekilde bölünmektedir.
